Windows Build Procedure

Download Microsoft Visual C++ Express. It's free but they make you register it otherwise it disables itself in 30 days. This registration is free but annoying as hell. Latest version known to work is 2008.

Link: Visual C++ Express

Download the precompiled boost libraries from Boost Consulting. Only versions > 1.33 will work. Latest version known to work is 1.36. When installing select the multithreaded static version.

Link: Boost

Download the latest gtkmm development installer. Change the destination folder to "C:\gtkmm" otherwise pkg-config won't work properly. Latest version known to work is 2.12.7-3. The 2.14 versions are known to have problems building.

Link: gtkmm

Download the production version of Python. Latest version known to work is 2.6.

Link: Python

Download the stable version of Scons. Latest version known to work is 1.0.1. After installation add C:\Python26\Scripts to your path.

Link: Scons

Open cmd.exe and navigate to the source directory. Run scons.
